# grathimut
GRAphic THings and IMproved outpUT (Grathimut) - это библиотека на C# для C#, которая даёт (будет давать в будущем) многочисленные способы вывода и кастомизации терминала: от необычного вывода до безумных эффектов. Использовать на своё усмотрение: хоть в шутку, хоть для улучшения своего TUI. Приглашаю каждого помогать в создании проекта!
